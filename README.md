# TaskMate – To-Do List App
TaskMate is a simple and efficient to-do list app built to help you organize tasks, stay focused, and boost your productivity. With an intuitive interface and responsive design, TaskMate makes task management quick and easy.

**Features**
Add, edit, and delete tasks easily.

Mark tasks as completed.

Responsive design for desktop and mobile users.

Lightweight and fast performance.

**Tech Stack**
Frontend: HTML, CSS, JavaScript (or React if used)

Version Control: Git & GitHub

Deployment: GitHub Pages
